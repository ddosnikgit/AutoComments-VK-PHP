# AutoComments-VK-PHP
## PHP script for posting comments in VK groups.
* Author - https://vk.com/ddosnik 
* Website - https://ddosnik.site/ 
## Installation guide: 
* Ubuntu/Debian: 
```
apt -y update && apt -y install php
bash start.sh
```
* Arch/Manjaro:
```
pacman -Syu && pacman -S php
bash start.sh
```
